# **App Name**: Knowledge Map3D

## Core Features:

- 3D Node Display: Display nodes in a 3D space arranged in a sphere.
- Link Visualization: Display links between nodes as lines in the 3D space.
- Search Highlighting: Highlight nodes and their connected links based on a search query.
- JSON Upload: Allow users to upload graph data in JSON format to define nodes and links.
- PDF Upload: Allow users to upload PDF files, which will be treated as new nodes in the graph, linked to a central node.
- Auto-Link Generator: Generative AI tool to generate relationships/links to enhance user's network of thought between uploaded files.

## Style Guidelines:

- Primary color: Deep sky blue (#3498db) for clarity and network visualization.
- Background color: Dark gray (#34495e) for contrast and to highlight the graph elements.
- Accent color: Hot pink (#f368e0) to highlight selected nodes or search results.
- Clean, sans-serif font for readability in the UI elements.
- Minimalist icons for file upload and search functionalities.
- Smooth transitions and animations for node highlighting and graph updates.
